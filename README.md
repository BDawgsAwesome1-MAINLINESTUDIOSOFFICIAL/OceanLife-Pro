# OceanLife Pro

Photorealistic beach simulation built with Three.js. Single-file HTML with loading screen, control panel, waves, sand, and ambient effects.

## Run

Open `index.html` in a modern browser (Chrome, Firefox, Safari, Edge).

## Features

- 3D beach scene with Three.js
- Wave animation and water effects
- Configurable controls (time of day, weather, etc.)
- Loading screen and UI panel
